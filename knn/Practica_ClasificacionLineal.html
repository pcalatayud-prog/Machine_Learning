Práctica de Clasificación Lineal: Regresión Logística¶

En esta prctica utilizaremos el dataset MNIST, compuesto por 40000 imgenes (de 28x28 pixeles) correspondientes a distintas versiones digitalizadas de los dgitos 0, ..., 9. El objetivo es clasificar correctamente cada una de las imgenes, utilizando el valor de los 784 pixeles. La informacin est disponible en el fichero 'train.csv'; la primera columna es la etiqueta del dgito (de 0 a 9) y el resto de columnas son los valores de los pixeles.
In [1]:
train <- read.csv('train.csv')
In [2]:
# Clase (dgito del 0 al 9 a predecir)
y <- train[,1]; 
# 784 columnas con los datos de los pixeles
x <- train[,-1]
In [3]:
# Cada dgito est igualmente representado en la muestra
hist(y,breaks = -0.5:9.5)
In [4]:
# Dibujamos los 6 primeros dgitos de la base de datos
par(mfrow = c(2,3))
for (i in 1:6) {
    image(matrix(as.matrix(x[i,1:784]), nrow = 28, ncol = 28))
    title(y[i])
}
Este problema se encuandra dentro de la clasificacin multi-categora (tenemos 10 posibles clases a predicir para cada dato). Para simplificar estos problemas, se suelen consideran tantos problemas binarios (predecir s o no) independientes como categoras se tengan. Es decir, un modelo para clasificar 0 (si o no), otro para 1, etc. En este ejemplo, consideramos el dgito '9' y el objetivo es predecir si una imagen es 9 o no.
In [5]:
# Seleccionamos un dgito concreto para tratar de predecirlo. Por ejemplo el nueve
# y9 = 1 si el dgito es 9 y 0 en caso contrario
y9 <- y;
y9[which(y != 9)] <- 0 ; y9[which(y == 9)] <- 1
In [6]:
# Construimos un dataframe para entrenar el modelo
dat <- data.frame(y9,x)
Como el tamao de la muestra es muy grande (40000 imagenes), seleccionamos una muestra para entrenar; por ejemplo, los 1000 primeros dgitos y el resto lo usaremos para validar.
In [7]:
i<- 1:1000;
#Random sampling
#i<-sample(1:dim(x)[1],1000);
datT <- dat[i,];
datt<- dat[-i,];
Aunque no es el modelo adecuado, construimos primero un modelo de regresin lineal para estos datos.

# Entrenamos primero un modelo lineal
model <- lm(y9~., data = datT)
out <- model$fitted.values
# El histograma muestra valores predichos (incluyendo negativos y mayores que 1)
hist(out)
